# CRM DataTable Standard

## Purpose

Defines the standard table implementation used across all CRM modules.

## Generated Assets

| Asset | Purpose |
|--------|---------|
| DataTable | Primary grid |
| Columns | Column definitions |
| Filters | Search and filtering |
| Toolbar | Bulk actions and commands |
| Validation | Shared schema/default values |
| Server Actions | CRUD server operations |

## Design Goals

- Consistent UX
- Reusable components
- Typed entities
- Server Action ready
- Hybrid SaaS / On-Prem compatible
- Automation-first generation

## Target Modules

- Companies
- Contacts
- Opportunities
- Tasks
- Calendar
- Activities
- Quotes
- Invoices
