# CRM Module Registry

## Purpose

Provides a centralized registry for all CRM modules generated by the Module Factory.

## Generated Manifest

| Property | Description |
|----------|-------------|
| id | Unique module identifier |
| name | Display name |
| route | CRM route |
| enabled | Feature toggle |
| icon | Navigation icon |
| version | Module version |

## Registry

`src/config/crm/modules.generated.ts`

The registry is automatically updated by the registration script.

## Benefits

- Dynamic navigation
- Feature flags
- Module discovery
- Hybrid SaaS compatibility
- On-Prem compatibility
- Future plugin support
